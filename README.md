# clock_user_bot

## Instalation
```bash
git clone https://github.com/fox-dever/clock_user_bot
cd clock_user_bot
pip3 install -r requrements.txt
```

## Setup
```bash
touch .env
echo "API_ID='<YOUR_API_ID>'\n" > .env
echo "API_HASH='<YOUR_API_HASH>'\n" > .env
chmod +x app.py
```

## Run
```bash
./app.py activate
```

## Stop
```bash
./app.py deactivate
```
